# TournamentController API

Controlador para gestión de torneos.

**Nota:** Actualmente todos los métodos están sin implementar.

## Endpoints

Todos los endpoints requieren autenticación (Sanctum) según las rutas configuradas.

### GET /api/tournaments
Listar todos los torneos.

**Autenticación:** Requerida (Sanctum)

**Estado:** ⚠️ No implementado

---

### POST /api/tournaments
Crear un nuevo torneo.

**Autenticación:** Requerida (Sanctum)

**Estado:** ⚠️ No implementado

---

### GET /api/tournaments/{id}
Obtener un torneo específico.

**Autenticación:** Requerida (Sanctum)

**Parámetros de ruta:**
- `id` (integer) - ID del torneo

**Estado:** ⚠️ No implementado

---

### PUT/PATCH /api/tournaments/{id}
Actualizar un torneo.

**Autenticación:** Requerida (Sanctum)

**Parámetros de ruta:**
- `id` (integer) - ID del torneo

**Estado:** ⚠️ No implementado

---

### DELETE /api/tournaments/{id}
Eliminar un torneo.

**Autenticación:** Requerida (Sanctum)

**Parámetros de ruta:**
- `id` (integer) - ID del torneo

**Estado:** ⚠️ No implementado
