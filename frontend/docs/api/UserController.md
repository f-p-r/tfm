# UserController API

Controlador para gestión de usuarios (CRUD completo).

**Nota:** Actualmente todos los métodos están sin implementar.

## Endpoints

Todos los endpoints requieren autenticación (Sanctum) según las rutas configuradas.

### GET /api/users
Listar todos los usuarios.

**Autenticación:** Requerida (Sanctum)

**Estado:** ⚠️ No implementado

---

### POST /api/users
Crear un nuevo usuario.

**Autenticación:** Requerida (Sanctum)

**Estado:** ⚠️ No implementado

---

### GET /api/users/{id}
Obtener un usuario específico.

**Autenticación:** Requerida (Sanctum)

**Parámetros de ruta:**
- `id` (integer) - ID del usuario

**Estado:** ⚠️ No implementado

---

### PUT/PATCH /api/users/{id}
Actualizar un usuario.

**Autenticación:** Requerida (Sanctum)

**Parámetros de ruta:**
- `id` (integer) - ID del usuario

**Estado:** ⚠️ No implementado

---

### DELETE /api/users/{id}
Eliminar un usuario.

**Autenticación:** Requerida (Sanctum)

**Parámetros de ruta:**
- `id` (integer) - ID del usuario

**Estado:** ⚠️ No implementado
