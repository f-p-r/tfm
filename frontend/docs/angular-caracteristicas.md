# Características de Angular ya usadas

- **Arranque sin NgModules**: `bootstrapApplication` con configuración global en `ApplicationConfig` para router, HTTP e interceptores, coalescing de eventos y listeners de errores globales ([src/main.ts](src/main.ts#L1-L6), [src/app/app.config.ts](src/app/app.config.ts#L1-L18)).
- **Enrutado avanzado**: rutas declarativas con `Routes`, lazy `loadComponent`, rutas anidadas, `canActivate` y wildcard de fallback ([src/app/app.routes.ts](src/app/app.routes.ts#L1-L101)).
- **Componentes standalone con OnPush**: componentes que declaran sus `imports` y usan `ChangeDetectionStrategy.OnPush` ([src/app/app.ts](src/app/app.ts#L1-L13), [src/app/components/navbar/navbar.component.ts](src/app/components/navbar/navbar.component.ts#L23-L178), [src/app/components/user-menu/user-menu.component.ts](src/app/components/user-menu/user-menu.component.ts#L8-L77)).
- **Señales (`signal`, `computed`, `effect`) para estado local y derivado**: título de la app, menús, filtros de juegos y permisos, apertura de dropdown, y caches locales ([src/app/app.ts](src/app/app.ts#L11-L12), [src/app/components/navbar/navbar.component.ts](src/app/components/navbar/navbar.component.ts#L30-L167), [src/app/components/user-menu/user-menu.component.ts](src/app/components/user-menu/user-menu.component.ts#L15-L75), [src/app/core/games/games.store.ts](src/app/core/games/games.store.ts#L14-L168), [src/app/core/context/context.store.ts](src/app/core/context/context.store.ts#L27-L114)).
- **Inyección con `inject()`**: servicios, guards y componentes obtienen dependencias sin constructores ([src/app/components/navbar/navbar.component.ts](src/app/components/navbar/navbar.component.ts#L38-L111), [src/app/core/auth/auth.service.ts](src/app/core/auth/auth.service.ts#L12-L79), [src/app/guards/game-by-slug.guard.ts](src/app/guards/game-by-slug.guard.ts#L27-L78), [src/app/core/associations/association-by-slug.guard.ts](src/app/core/associations/association-by-slug.guard.ts#L18-L55)).
- **Limpieza automática de suscripciones**: `takeUntilDestroyed` en navegación y listeners del DOM para evitar fugas ([src/app/components/navbar/navbar.component.ts](src/app/components/navbar/navbar.component.ts#L59-L79), [src/app/components/user-menu/user-menu.component.ts](src/app/components/user-menu/user-menu.component.ts#L34-L50)).
- **Guards funcionales `CanActivateFn`**: validación de slugs, carga perezosa de datos y actualización de contexto antes de activar rutas ([src/app/guards/game-by-slug.guard.ts](src/app/guards/game-by-slug.guard.ts#L27-L78), [src/app/core/associations/association-by-slug.guard.ts](src/app/core/associations/association-by-slug.guard.ts#L18-L55)).
- **HTTPClient + interceptores funcionales**: `provideHttpClient` con `withInterceptors` para credenciales y cabecera XSRF, más servicios de API tipados ([src/app/app.config.ts](src/app/app.config.ts#L13-L16), [src/app/core/interceptors/credentials.interceptor.ts](src/app/core/interceptors/credentials.interceptor.ts#L1-L14), [src/app/core/interceptors/xsrf.interceptor.ts](src/app/core/interceptors/xsrf.interceptor.ts#L1-L30), [src/app/core/auth/auth.service.ts](src/app/core/auth/auth.service.ts#L17-L79), [src/app/core/associations/associations-api.service.ts](src/app/core/associations/associations-api.service.ts#L1-L43)).
- **Caches en memoria con TTL**: resolución de asociaciones y catálogo de juegos con deduplicación de peticiones y expiración temporal ([src/app/core/associations/associations-resolve.service.ts](src/app/core/associations/associations-resolve.service.ts#L26-L179), [src/app/core/games/games.store.ts](src/app/core/games/games.store.ts#L10-L168)).
- **Gestión de contexto global**: `ContextStore` como señal compartida para scope global/juego/asociación, usada por guards y stores ([src/app/core/context/context.store.ts](src/app/core/context/context.store.ts#L27-L114), [src/app/guards/game-by-slug.guard.ts](src/app/guards/game-by-slug.guard.ts#L27-L78), [src/app/core/associations/association-by-slug.guard.ts](src/app/core/associations/association-by-slug.guard.ts#L18-L55), [src/app/core/games/games.store.ts](src/app/core/games/games.store.ts#L39-L51)).
